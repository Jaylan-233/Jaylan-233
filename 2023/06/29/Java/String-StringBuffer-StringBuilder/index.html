String StringBuffer StringBuilder

为什么要区分这么多的字符串类

字符串是使用最为频繁的数据类型之一

有的字符串需要经常修改 、有的的不需要

有时需要处理大量字符、有时只要处理少量的字符串

根据操作的不同特点 使用不同的的字符串类型

共同点

  • 都是字符串类
  • 都是用来处理字符串的

不同点

执行速度

StringBuilder >StringBuffer>String

安全性

String -线程安全

StringBuffer -线程安全

StringBuilder-线程不安全

适用范围

  • 大量

    • 单线程-StringBuilder
    • 多线程-StringBuffer

  • 少量

    • String
