SpringBoot常见错误

发表于 2023-06-18

Failed to configure a DataSource: ‘url’ attribute is not specified and no embedded datasource could be configured.

翻译

配置数据源失败:没有指定’url’属性，无法配置任何嵌入的数据

❌原因

项目使用了数据库 但是给其指定 相关信息 url 用户名 密码啥的

✔️解决办法

修改配置文件(application.properties) 加上数据库相关的 信息

  • 数据库的驱动(记得看下版本 不同版本数据库的加载驱动不同)
  • 数据库的url(记得设置时区)
  • 数据库用户名(默认都是root)
  • 数据库密码

小结

  • 没有在配置文件中设置数据库的相关信息 导致找不到数据库

SQL dialect is not configured.

翻译:

没有配置SQL方言(方言?)

没有告诉IDEA SQL是哪个数据软件的 导致其无法对此进行检查

❌原因

IDEA中没设置SQL的语言 导致SQL检查失败

✔️解决办法

到设置中指定SQL语言

Unable to resolve table xxx

情况: SQL语句没问题 运行没问题 但IDEA 给出警告和错误 问题出现在检查上

另外还有

No data sources are configured to run this SQL and provide advanced code assistance. Disable this inspection via problem menu (Alt+Enter).

翻译

无法解析表“xxx”

没有配置任何数据源来运行此SQL并提供高级代码帮助。通过问题菜单(Alt+Enter)禁用此检查。

❌原因:

IDEA 没能连接到该数据库 所以没办法进行解析以及 其他操作

✔️解决办法:

设置IDEA中的 Database工具 将其连接到要进行操作的数据库

连接了正确的数据库之后 错误就会消失 反之 导入的数据库可能不正确
